# HTML-Cleaner

A Python script that purifies HTML content by removing style-related elements, scripts, URL links, and media tags while preserving the HTML structure and data content.

## Features

The HTML Cleaner removes:
- All style-related tags (`<style>`, `<link rel="stylesheet">`)
- All style attributes from HTML elements
- All script-related tags (`<script>`)
- All href attributes from links
- All media tags (`<img>`, `<video>`, `<audio>`, `<source>`, `<track>`, `<embed>`, `<object>`, `<iframe>`)

While preserving:
- HTML document structure
- Text content and data
- Other attributes (class, id, type, name, etc.)
- Semantic HTML tags

## Files

The project contains 3 main files:

1. **input.html** - Sample HTML file with content to be cleaned
2. **html_cleaner.py** - Python script that purifies the HTML
3. **output.html** - Cleaned HTML output (generated by the script)

## Usage

### Basic Usage

Simply run the script to clean the default input.html file:

```bash
python3 html_cleaner.py
```

This will read from `input.html` and write the cleaned HTML to `output.html`.

### Custom Input/Output Files

You can specify custom input and output files:

```bash
python3 html_cleaner.py <input_file> <output_file>
```

Example:
```bash
python3 html_cleaner.py my_input.html my_output.html
```

## Requirements

- Python 3.x (uses only standard library modules)

No external dependencies are required.

## Example

**Input (input.html):**
```html
<div style="color: red;">
    <h1 style="font-size: 24px;">Hello World</h1>
    <a href="https://example.com">Link</a>
    <img src="image.jpg" alt="Image">
    <script>console.log('test');</script>
</div>
```

**Output (output.html):**
```html
<div>
    <h1>Hello World</h1>
    <a>Link</a>
    
    
</div>
```

## How It Works

The script uses Python's built-in `html.parser.HTMLParser` to parse the HTML and selectively remove unwanted tags and attributes while maintaining the document structure. It processes the HTML sequentially, filtering out unwanted elements during the parsing phase.
